class babylon.utils.TempVars

Available on all platforms

Temporary variables . Engine classes may access these temp variables with TempVars.getTempVars(), all retrieved TempVars instances must be returned via TempVars.release(). This returns an available instance of the TempVar class ensuring this particular instance is never used elsewhere in the mean time.
